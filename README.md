## 🌍 Project Overview

GreenCampus AI is an AI-powered sustainability monitoring and decision-support platform designed to help educational institutions identify, monitor, and address sustainability-related issues on campus.

The system enables users to report environmental issues, analyze sustainability trends, visualize impact metrics, and obtain AI-driven recommendations for improving campus sustainability.

This project was developed as part of the **1M1B – IBM SkillsBuild AI for Sustainability Virtual Internship**.

---

## 🎯 Problem Statement

Educational institutions often face challenges related to:

* 💧 Water wastage and leakages
* ⚡ Excessive energy consumption
* ♻ Improper waste management
* 📊 Lack of sustainability monitoring
* 🤖 Limited decision-support systems

### Proposed Solution

Develop an AI-enabled platform that:

* Monitors sustainability issues
* Categorizes complaints automatically
* Prioritizes critical issues
* Generates sustainability insights
* Supports decision-making through AI recommendations

## 🚀 Key Features

### 📝 Issue Reporting System

Users can report:

* Water-related issues
* Energy-related issues
* Waste-management issues
* Other sustainability concerns

---

### 🤖 AI-Based Issue Classification

The system automatically classifies issues into:

* 💧 Water
* ⚡ Energy
* ♻ Waste
* 📌 Other

using AI-inspired text analysis techniques.

---

### 🚨 Smart Priority Detection

Issues are categorized as:

* 🔴 High Priority
* 🟡 Medium Priority
* 🟢 Low Priority

based on their severity and impact.

---

### 📊 Sustainability Dashboard

Interactive dashboard featuring:

* Total Issues
* Water Issues
* Energy Issues
* Waste Issues
* Green Campus Score
* Sustainability Impact Metrics
* Issue Distribution Visualization
* AI Recommendations

---

### 🌱 Green Campus Score

The platform evaluates campus sustainability performance and generates a dynamic sustainability score.

---

### 📈 Impact Metrics

Estimated:

* 💧 Water Saved
* ⚡ Energy Conserved
* ♻ Waste Recycled

based on reported sustainability activities.

---

### 🤖 AI Sustainability Assistant

An AI-powered assistant that answers sustainability-related questions and provides recommendations using a retrieval-based knowledge system.

Example Questions:

* How can campuses conserve water?
* How can energy consumption be reduced?
* What are sustainable waste management practices?
* How do SDGs support sustainability?

## ⚙ Installation

### Clone Repository

```bash

cd AI_for_Sustainibility
```

### Install Dependencies


```bash

```

